## Install
```
npm install -g baobab-ui --save
```
or

```
yarn add baobab-ui
```

## Usage
```
import Vue from 'vue'
import BaobabUI from 'baobab-ui'
import 'baobab-ui/lib/baobab-ui.css'
Vue.use(BaobabUI)
```
